# Admin MFE

Admin panel microfrontend (React 19).

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Module Federation

Exposes:
- `./RBACManagement` - Role and permission management
- `./UserManagement` - User management
- `./SystemSettings` - System settings

## Dependencies

- `@task-management/shared-ui` - Shared UI components
