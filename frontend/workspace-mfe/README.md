# Workspace MFE

Workspace and project management microfrontend (React 19).

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Module Federation

Exposes:
- `./KanbanBoard` - Kanban board component
- `./TaskList` - Task list component
- `./ProjectManagement` - Project management component
- `./WorkflowDesigner` - Workflow designer component

## Features

- Kanban board with drag & drop (@dnd-kit)
- Real-time updates via WebSocket
- React 19 features (useOptimistic, useActionState)

## Dependencies

- `@task-management/shared-ui` - Shared UI components
