# Shared UI Components

Shared React 19 component library for microfrontends.

## Installation

```bash
npm install
npm run build
```

## Publishing

```bash
npm login
npm publish --access public
```

## Usage

```bash
npm install @task-management/shared-ui
```

```typescript
import { Button, Input, Card } from '@task-management/shared-ui';
import { useAuth, useApi } from '@task-management/shared-ui';
```

## Components

- Button
- Input
- Modal
- Card

## Hooks

- useAuth
- useApi
- useWebSocket
