# Auth MFE

Authentication microfrontend built with React 19.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Module Federation

Exposes:
- `./Login` - Login component
- `./Register` - Register component
- `./Profile` - Profile management component

## Dependencies

- `@task-management/shared-ui` - Shared UI components
