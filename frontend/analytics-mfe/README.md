# Analytics MFE

Analytics and reporting microfrontend (Next.js 14+).

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Module Federation

Exposes:
- `./Dashboard` - Analytics dashboard
- `./Reports` - Report generation

## Features

- Server-side rendering
- Charts with Recharts
- Report generation UI

## Dependencies

- `@task-management/shared-ui` - Shared UI components
