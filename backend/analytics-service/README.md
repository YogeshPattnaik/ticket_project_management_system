# Analytics Service

SpringBoot service for analytics and reporting with Elasticsearch integration.

## Features
- Report generation
- Metrics calculation
- Data export
- Elasticsearch integration for advanced search

## Configuration
- Elasticsearch cluster configuration
- PostgreSQL connection for data aggregation
- Report templates
- Export formats (PDF, CSV, Excel)

## API Endpoints
- GET /api/v1/analytics/reports
- POST /api/v1/analytics/reports/generate
- GET /api/v1/analytics/metrics
- GET /api/v1/analytics/export
